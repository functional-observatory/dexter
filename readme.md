## Prerequisites

- (`dotnet sdk`)[https://dotnet.microsoft.com/download]
- (`node`)[https://nodejs.org/en/]
- (`yarn`)[https://yarnpkg.com/]

## Getting Started

- Clone the repo.
- Install dependencies.

```sh
yarn
dotnet paket update
```

- Start development server with hot reload.

```sh
yarn dev
```

- Open (http://localhost:4500/)[http://localhost:4500/]
